# Blakhole_UI 功能完成度清单
> 更新时间: 2026-07-10
> 验证方式: 对照 QML 控件 -> BlackHoleCore 属性 -> 配置文件 -> BlackholeConfig -> gui_config.cpp 读取 -> main.cpp 运行时 uniform/shader 逻辑，并执行编译与 shader 拼接检查。

## 标记说明

| 标记 | 含义 |
|------|------|
| 已端到端 | 设置写入配置文件，`blackhole.exe` 读取并在运行时生效 |
| UI 内部闭环 | 设置只由 Qt UI 自身处理，不需要 `blackhole.exe` |
| 预留/实验 | 代码保留或实验路径存在，但默认不作为稳定功能入口 |

## 一、端到端可用

### 预设与基础渲染参数

| 功能 | 配置文件 | 后端使用方式 |
|------|----------|--------------|
| 16 预设 x 14 参数 | `blackhole_presets.txt` | `uPresetTemp/uPresetIncl/...` 数组 uniform |
| 始终显示 / 空闲检测 | `blackhole_presets.txt` | `cfg.mode` 控制 monitor 是否启动 renderer |
| 空闲秒数 | `blackhole_presets.txt` | `cfg.idleSec` |
| 播放模式: 顺序/循环/随机 | `blackhole_presets.txt` | `uPlayMode` |
| 切换间隔 | `blackhole_presets.txt` | `uSlotSec` |
| 固定大小 | `blackhole_presets.txt` | `uFixedSize/uFixedLevel` |
| 捕获方式: 自动/WGC/DXGI | `blackhole_presets.txt` | `cfg.captureMode` 选择 WGC/DXGI |
| 多显示器选择 | `blackhole_presets.txt` | `cfg.displayMode` 选择主屏/副屏/跨屏/一屏一黑洞 |

### 高级效果

| 功能 | 配置键 | 后端使用方式 |
|------|--------|--------------|
| 黑洞大小 | `holeSize` / `holeRadius` | 映射到 `uHoleRadius` |
| 动画速度 | `animationSpeed` / `spd` | 映射到 `uSpeed` |
| 路径随机化开关 | `randomPath` | 控制启动时 home/phase/presetOffset 是否随机 |
| 跟随鼠标 | `followMouse` | 每帧读取 `GetCursorPos()`，转换为 `uHomeX/uHomeY` |
| 逐渐增长开关 | `growEnabled` | 控制 shader 增长曲线 |
| 初始大小 | `initialSize` | `growEnabled=true` 时作为增长起点 |
| 扭曲程度 | `distortion` | 映射到 `uDistortion`，调整远场/近场透镜位移强度 |
| 吞噬屏幕 UI 特效 | `screenSwallow` | 映射到 `uScreenSwallow`，绑定出生/退出动画的轻微径向压暗 |

> UI 保存链路说明: `holeSize`、`distortion`、`growEnabled` 等高级设置已确认从 `AdvancedConfig.qml` 写回 `BlackHoleCore`，再由 `saveAdvancedConfig()` 持久化到 `blackhole_advanced.txt`。
