# v1.2.3 设置保存修复设计

## 目标

发布 `v1.2.3`，修复“槽位秒数”无法手动输入和多屏模式重启后恢复默认的问题，并将界面名称改为“黑洞变换秒数”。

## 根因与修复

- `BlackholeConfig.qml` 的小数 SpinBox 只有 `textFromValue`，缺少 `valueFromText`。补充文本到十分之一秒整数值的转换，保持现有 `0.5-60.0` 秒范围。
- `BlackHoleCore::loadConfig()` 读取配置行最后一个 `screenTarget` 字段后用 `atEnd()` 拒绝赋值。改为依据 `QTextStream::status()` 判断读取是否成功，兼容旧配置缺少该字段时默认主屏。
- 多屏模式继续由 `saveConfig()` 在启动、保存和退出时写入，不改变现有配置格式。

## 验证

- 新增静态回归契约，验证中文名称、`valueFromText`、行末字段状态判断和旧错误逻辑消失。
- 运行现有 QML/Core 契约、Qt CTest、Renderer/UI Release 构建。
- 同步唯一版本源、README、发布脚本和检查脚本到 `1.2.3`，生成 ZIP 与 SHA-256 sidecar。
